StudyFlow – Smart Study Planner
StudyFlow is a smart study planner web application designed to help students organize their learning, track progress, and stay consistent with their studies.
It provides tools like study planning, notes management, file uploads, topic guidance, and a focus timer, all in one place.

🚀 Features
📅 Study Planner
Create a structured study plan by selecting:
Course
Subject
Topic
The app generates a study plan with guidance and recommendations.
🧠 Topic Guidance
Each topic provides:
Summary
Difficulty level
Estimated study time
Recommended books
This helps students understand what to study and how difficult it is.
📝 Topic Notes
Students can:
Write their own notes
Save notes for later review
View saved notes anytime
Notes are stored using Local Storage, so they remain even after refreshing the page.
📂 Study Material Upload
Users can upload:
PDFs
Documents
Study resources
This allows students to keep all learning materials in one place.
📊 Study Progress Tracker
Students can mark completed topics and track their progress through a progress bar system.
⏱ Focus Timer
Includes a study timer to help students stay focused while studying.
📚 Book Recommendations
The system suggests useful textbooks related to the selected topic.
Example:
Basic Electrical Engineering – VK Mehta
Engineering Circuit Analysis – Hayt
Electrical Technology – BL Theraja
🛠 Technologies Used
HTML5
CSS3
JavaScript (DOM Manipulation)
Local Storage API
